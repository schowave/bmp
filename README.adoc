= Bundesliga Manager Professional

image::images/bmp.png[Bundesliga Manager Professional]

Containerized DOS classic — playable in the browser via noVNC.

== Development

=== Prerequisites

* https://podman.io[Podman] (or Docker)

=== Run locally

[source,bash]
----
make run
----

Opens at http://localhost:8080. Savegames persist in `./savegame/`.

[source,bash]
----
make stop
----

=== Makefile targets

[cols="1,3"]
|===
| `make build` | Build the container image
| `make run`   | Stop, build, and start in detached mode
| `make stop`  | Stop and remove the container
| `make push`  | Build and push multi-arch image to Docker Hub
|===

== Release

=== Via GitHub Actions (recommended)

1. Go to **Actions** → **Release** → **Run workflow**
2. Leave version empty for auto-increment (e.g. `0.1.0` → `0.1.1`), or enter a specific version
3. The workflow will:
   - Update the `VERSION` file and commit
   - Create a git tag `v0.1.1`
   - Build a multi-arch image (amd64 + arm64)
   - Push `schowave/bmp:0.1.1` and `schowave/bmp:latest` to Docker Hub
   - Create a GitHub Release with auto-generated notes

Required GitHub secrets: `DOCKERHUB_USERNAME`, `DOCKERHUB_TOKEN`

=== Manual push

[source,bash]
----
make push
----

Prompts for version, builds multi-arch, and pushes to Docker Hub.

== Deployment

=== Synology NAS

1. Create a project folder on your NAS (e.g. `/volume1/docker/bmp/`)
2. Add the `docker-compose.yml` from this repository
3. In **Container Manager** → **Project** → **Create**, point to the folder and start
4. If using a reverse proxy: add WebSocket headers in **Reverse Proxy** → **Benutzerdefinierte Kopfzeile**:
+
[cols="1,1"]
|===
| `Upgrade` | `$http_upgrade`
| `Connection` | `$connection_upgrade`
|===

The included https://containrrr.dev/watchtower/[Watchtower] service monitors Docker Hub and automatically updates the container when a new release is published.

=== Other Platforms

The Docker image `schowave/bmp` is built for `linux/amd64` and `linux/arm64`. It works on any platform that supports Docker or Podman.

[source,yaml]
----
# docker-compose.yml
services:
  bmp:
    image: schowave/bmp:latest
    ports:
      - "8080:80"
    volumes:
      - ./savegame:/savegame
    restart: unless-stopped
----
